## Functional:

Oперации: 

`+` -> Оператор сложения; <br>
`-` -> Оператор вычитания; <br>
`*` -> Оператор умножения; <br>
`/` -> Оператор деления; <br>
`+/-` -> Изменение числа на положительное/отрицательное; <br>
`=` -> Вывод результата вычисления; <br>
`с` -> Удаляет введенное значение/результат.
